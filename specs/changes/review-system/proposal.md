# 提案：顧客評價系統 (Customer Review System)

## 1. 目標 (Goals)
建立一個完整的評價系統，讓顧客在完成預約後能夠分享他們的真實體驗。這不僅能為其他顧客提供參考，也能幫助美容師了解自己的服務表現，並提升平台的整體服務品質與信任度。

## 2. 核心功能 (Core Features)
- **評價提交**：顧客可對已完成的預約進行星級評分 (1-5) 與文字留言。
- **預約狀態整合**：只有狀態為「已完成」(confirmed 且時間已過) 的預約可進行評價。
- **一 booking 一評**：確保每筆預約只能被評價一次，防止數據灌水。
- **美容師數據聚合**：自動更新美容師的平均評分與總評價數。
- **評價展示**：在美容師個人頁面展示所有評價歷程。

## 3. 預期效益 (Expected Benefits)
- **增加信任感**：透過真實評價吸引新顧客。
- **提升品質**：反饋機制促使美容師維持高水準服務。
- **互動性**：增強顧客與平台的互動，提升黏著度。

## 4. 衡量指標 (Metrics)
- 預約完成後的評價率 (Review Rate)。
- 平台平均評分的變化趨勢。
- 評價內容的質量與真實性。
